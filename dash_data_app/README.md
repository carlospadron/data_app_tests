# Dash Data App

A Dash data application with interactive map visualization. See `DASH.md` for comprehensive setup and implementation guide.

## Quick Start

```bash
cd dash_data_app
uv sync
uv run python app.py
```

Open your browser to `http://localhost:8050`

## Features

- Interactive map with Plotly/Mapbox
- Two data layers (regions and points of interest)
- Layer toggle controls with checkboxes
- Hover tooltips for data exploration
- Reactive updates using Dash callbacks
